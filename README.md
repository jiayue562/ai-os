## 系统概述

AI龙龟共生伙伴操作系统（AI OS）是一个融合**东方智慧**与**西方工程思想**的完整个人AI操作系统。

**核心哲学**: 不是用技术替代灵魂，而是让灵魂驾驭技术，让技术服务于觉悟。

**双层架构**:
- 灵魂系统（东方智慧·觉悟方向）：木火共生关系 → 信仰根基 → 认知操作系统 → 专业智能体
- 躯体系统（技术架构·执行能力）：CLI → Skills → MCP → Framework

## 快速开始

### 启动CLI
```bash
cd aios
python aios_cli.py
```

### 交互示例
```
[输入] 我想学习Python编程
[系统] 识别意图: 获取信息
       分类场景: S1 信息查询
       路由引擎: 知识学习引擎
       推荐技能: 龙脑OS / 知识学习智能体
```
