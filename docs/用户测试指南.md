## 基本使用方法

### 1. 启动系统
系统启动后，你会看到欢迎界面：
```
======================================
AI 龙龟共生伙伴操作系统 (AI OS)
版本: 1.0.0 (WorkBuddy集成版)
用户: 悟空 (编程小白)
助手: 龙龟神将
======================================
```

### 2. 输入自然语言请求
直接用中文告诉我你的需求，例如：
- "我想学习Python编程"
- "分析团队协作问题"
- "创建一个AI助手项目计划"
- "分析我的性格特点"
- "龙心OS需要升级优化"

### 3. 查看系统响应
系统会自动：
1. **识别意图**：分析你想做什么
2. **分类场景**：判断属于哪种场景（S0-S9）
3. **路由引擎**：选择合适的处理引擎
4. **推荐技能**：推荐最相关的技能帮你解决问题
5. **生成回答**：用自然语言回复你

### 4. 常用命令
- `帮助` 或 `help` - 查看帮助信息
- `状态` 或 `status` - 查看系统状态
- `历史` 或 `history` - 查看对话历史
- `清除` 或 `clear` - 清除对话历史
- `退出` 或 `quit` - 退出系统

**解决方案**：确保在正确的目录中运行，或尝试：
